# Video Editor fixed files

Upload every file in this folder to the same GitHub Pages folder/repository root:

- index.html
- styles.css
- app.js
- logo.png
- ffmpeg.min.js
- ffmpeg-core.js
- ffmpeg-core.wasm
- ffmpeg-core.worker.js

Do not open `index.html` directly from your computer as a `file://` URL. Use GitHub Pages or run a local server, for example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
